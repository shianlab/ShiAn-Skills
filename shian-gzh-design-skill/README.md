# shian-gzh-design · Shian 公众号排版 Skill

Shian 的个性化微信公众号排版工具。它把 Markdown 整理为可直接复制进微信公众号编辑器的内联样式 HTML，重点服务内容表达、移动端阅读和稳定复用。

## 设计原则

- **内容优先**：先保证标题层级、正文节奏和重点清晰，再做视觉装饰。
- **移动端优先**：正文基准字号不低于 `17px`，行高约 `1.85`，避免密、挤、小。
- **克制排版**：不自动添加大圆角标题卡、空洞口号、阅读地图或无意义装饰。
- **结构明确**：二级标题、数字序号、关键词加粗与细下划线构成主要阅读路标。
- **少量辅助框**：只在结论、提醒、步骤或引用确实需要承载时使用，并保持小圆角或直角。
- **身份干净**：不写入固定作者姓名、他人署名或联名信息；作者区仅依据文章原文生成。

## 默认主题

| 主题 | 气质 | 适合内容 |
|---|---|---|
| 青瓷绿稿纸 | 青瓷绿 + 陶土棕，清淡稳定 | AI 方法论、知识整理、效率复盘 |
| 暖杏手记 | 暖杏 + 陶橘，温和有人味 | 个人观察、成长复盘、生活记录 |
| 铅笔灰稿纸 | 铅笔灰 + 旧金，理性克制 | 产品分析、工具评测、系统说明 |
| 苔绿砂纸 | 苔绿 + 砂陶，沉稳耐读 | 案例复盘、长期主义、认知文章 |
| 珊瑚米白 | 低饱和珊瑚 + 玫瑰灰 | 轻观点、个人品牌、解释型文章 |

浅蓝网格莫兰迪仅作备用主题；只有用户明确要求浅蓝网格或蓝紫重点色时才使用。

## 使用方式

直接告诉 Codex：

```text
用 shian-gzh-design 排版这篇公众号文章，按内容自动选择主题。
```

也可以指定主题：

```text
用 shian-gzh-design 的「铅笔灰稿纸」排版这篇文章，输出可预览 HTML。
```

## 输入与输出

最少只需要提供一篇 Markdown。可选信息包括主题、作者署名、图片位置和输出目录。

默认输出：

1. 一份公众号内联样式 HTML。
2. 一份本地预览 HTML。
3. 图片缺失时保留清晰的待补图位置，不伪造素材。

## 文件入口

- 执行规则：`SKILL.md`
- 主题选择：`references/theme-index.md`
- Shian 编辑规范：`references/shian-editorial-system.md`
- HTML 校验：`scripts/validate_gzh_html.py`
- 组件校验：`scripts/component_lint.py`

## 维护约定

这是 Shian 的主公众号排版 Skill。正常情况下优先调用本 Skill；其他公众号或社交卡片 Skill 仅在用户明确点名、需要不同视觉语言或特殊输出形式时使用。
