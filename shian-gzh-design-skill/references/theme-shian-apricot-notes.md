# 公众号排版主题 —— 暖杏手记

> Shian 内容表达型主题之一。用于个人观察、轻观点、成长复盘、生活化 AI 使用记录。气质更温暖，有手记感，但不做卡片和贴纸。

## 组件规则

先读 `references/shian-editorial-system.md`，再读 `references/theme-sky-grid-morandi.md` 的 HTML 组件结构。使用同类组件，但必须按本主题的叙事节奏调整留白和章节识别，不能只换色。颜色替换如下：

- `#6FA8DC` → `#D7A06B`
- `#8E7CC3` / `#7E6BB6` / `#6F63A7` → `#C56F58`
- `#BFD7F1` → `#EBCBAA`
- `rgba(191,215,241,0.42)` → `rgba(225,190,146,0.30)`
- `#F7FBFF` / `#EEF6FF` → `#FFF5E8`
- `#F3F0FA` → `#FBECE6`
- `#4F545C` → `#5B5550`
- `#666B73` → `#6D665F`

## 适用节奏

- 适合温和、有个人表达的文章。
- 重点词是陶橘色，不用蓝紫。
- 辅助框像“手记旁注”，不要变成大圆角卡片。
