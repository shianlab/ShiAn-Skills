## 组件规则

先读 `references/shian-editorial-system.md`，再读 `references/theme-sky-grid-morandi.md` 的 HTML 组件结构。使用同类组件，但必须按本主题的理性节奏调整信息密度和章节识别，不能只换色。颜色替换如下：

- `#6FA8DC` → `#767B82`
- `#8E7CC3` / `#7E6BB6` / `#6F63A7` → `#B89B55`
- `#BFD7F1` → `#D8D0B5`
- `rgba(191,215,241,0.42)` → `rgba(190,194,198,0.30)`
- `#F7FBFF` / `#EEF6FF` → `#F4F5F5`
- `#F3F0FA` → `#F5F0E2`
- `#4F545C` → `#4C4F54`
- `#666B73` → `#666A70`

## 适用节奏

- 适合产品、技术、系统性说明。
- 读感更克制，重点色不跳。
- 避免大面积黑块，仍然保持内容型公众号节奏。
