> Shian 内容表达型主题之一。用于知识整理、AI 方法论、个人效率、复盘类文章。气质是清淡、稳定、耐读，避开参考图里的浅蓝 + 蓝紫组合。

## 组件规则

先读 `references/shian-editorial-system.md`，再读 `references/theme-sky-grid-morandi.md` 的 HTML 组件结构。使用同类“标题、章节、正文、重点词、左竖线辅助框”组件，但必须按本主题的版式节奏调整留白和章节识别，不能只换色。颜色替换如下：

- `#6FA8DC` → `#7A9D8C`
- `#8E7CC3` / `#7E6BB6` / `#6F63A7` → `#A56F54`
- `#BFD7F1` → `#CADBCD`
- `rgba(191,215,241,0.42)` → `rgba(186,207,195,0.34)`
- `#F7FBFF` / `#EEF6FF` → `#F2F7F3`
- `#F3F0FA` → `#F6EEE9`
- `#4F545C` → `#4F5A55`
- `#666B73` → `#666F69`

## 适用节奏

- 默认不加导读卡和圆框。
- 适合“讲清楚一个概念 / 方法 / 复盘”的稳态文章。
- 辅助框用青瓷绿左竖线，重点词用陶土棕，避免和浅蓝参考图撞色。
