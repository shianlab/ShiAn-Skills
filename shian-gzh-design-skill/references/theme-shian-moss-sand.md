## 组件规则

先读 `references/shian-editorial-system.md`，再读 `references/theme-sky-grid-morandi.md` 的 HTML 组件结构。使用同类组件，但必须按本主题的沉稳节奏调整章节间距和辅助块，不能只换色。颜色替换如下：

- `#6FA8DC` → `#6F8068`
- `#8E7CC3` / `#7E6BB6` / `#6F63A7` → `#B07A57`
- `#BFD7F1` → `#D4DEC6`
- `rgba(191,215,241,0.42)` → `rgba(190,198,177,0.32)`
- `#F7FBFF` / `#EEF6FF` → `#F3F6EF`
- `#F3F0FA` → `#F4EADF`
- `#4F545C` → `#50594D`
- `#666B73` → `#666D61`

## 适用节奏

- 适合复盘、案例、认知类内容。
- 主色偏自然，不像浅蓝参考图。
- 重点词用砂陶色，视觉更成熟。
