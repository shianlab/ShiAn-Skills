> Shian 内容表达型主题之一。用于轻观点、个人品牌、创作复盘、面向大众的解释型文章。比暖杏更明亮，但仍然低饱和。

## 组件规则

先读 `references/shian-editorial-system.md`，再读 `references/theme-sky-grid-morandi.md` 的 HTML 组件结构。使用同类组件，但必须按本主题的亲和节奏调整段落长度和章节识别，不能只换色。颜色替换如下：

- `#6FA8DC` → `#D9827C`
- `#8E7CC3` / `#7E6BB6` / `#6F63A7` → `#9B6F88`
- `#BFD7F1` → `#F0C9C2`
- `rgba(191,215,241,0.42)` → `rgba(226,178,168,0.28)`
- `#F7FBFF` / `#EEF6FF` → `#FFF1EC`
- `#F3F0FA` → `#F7EAF0`
- `#4F545C` → `#5B5353`
- `#666B73` → `#6C6464`

## 适用节奏

- 适合轻观点、生活化 AI 使用、创作方法。
- 色彩更有人味，但不做可爱贴纸风。
- 重点词用玫瑰灰，不使用蓝紫。
