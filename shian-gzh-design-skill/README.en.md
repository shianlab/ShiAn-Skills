# shian-gzh-design · Shian WeChat Formatting Skill

Shian's personalized formatter for turning Markdown into inline-styled HTML that can be pasted into the WeChat Official Account editor.

## Principles

- Content first: hierarchy, rhythm and emphasis come before decoration.
- Mobile first: body text is at least `17px`, with a line height around `1.85`.
- Restrained layout: no automatic oversized rounded title cards, slogans or decorative reading maps.
- Clear structure: headings, numbered sections, bold text and subtle underlines guide reading.
- Clean identity: never inject a fixed author name, third-party attribution or collaboration statement.

## Default themes

1. Shian Sage Paper
2. Shian Apricot Notes
3. Shian Graphite Pencil
4. Shian Moss Sand
5. Shian Coral Cream

The light-blue grid theme is retained only as an optional fallback when explicitly requested.

## Usage

```text
Use shian-gzh-design to format this WeChat article and choose the theme from its content.
```

See `SKILL.md` for the workflow and `references/theme-index.md` for theme selection.
