## 这里是什么

这里是 **时安 / shian** 的 Agent Skills 仓库。

我是一名 **AI 应用开发者和独立开发者**，平时围绕 AI 应用、Agent、Skill、知识库、自动化工作流和内容表达做项目，也持续把实践中真正有效的方法整理成可以复用的工具。

这个仓库就是这些实践的开放入口。这里不做一份冗长的 Skill 清单，也不逐个重复介绍功能——仓库目录本身就是最直接的索引。看到适合自己的内容，点进去读说明、下载、安装，然后让它进入你的工作流。

## 欢迎取用

如果其中某个 Skill 正好解决了你的问题，欢迎直接使用。

你可以下载、安装、Fork，也可以在原有基础上继续调整。这个仓库会随着我的项目和实践持续更新，新的方法、踩过的坑和更稳定的工作流，也会逐步整理进来。

具体用法、依赖与许可，请以各 Skill 目录内的说明为准。如果它确实帮到了你，也欢迎点一个 Star，让我知道这些整理值得继续。

我是 **时安 / shian**，一个 00 后、OPC、AI 应用开发者。
